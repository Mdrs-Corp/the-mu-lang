# Loq
Log the content to the user, in the terminal, whenever the type of the arguments
given:
```xml
<loq>
	||Hi, ||
	||my name is ||
	my_name
	|| and i have ||
	brothers_number
	|| brothers ||
</loq>
```
If my_name is 'jean' and brothers_number is 12, this part will print :
```xml
Hi my name is jean and i have 12 brothers
```
